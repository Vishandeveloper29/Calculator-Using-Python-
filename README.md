# 🧮 Python Terminal Calculator

A simple calculator program built with **Python** that runs in the **terminal / command prompt**.  
It performs basic arithmetic operations and shows a **large CALCULATOR banner** when the program starts.

---

## 🚀 Features

- ➕ Addition
- ➖ Subtraction
- ✖ Multiplication
- ➗ Division
- 🧩 Modulus (%)
- ⚠ Division by zero protection
- 🎨 Large ASCII title in terminal
- 💻 Simple CLI interface

---

## 📂 Project Structure

```
calculator-project
│
├── calculator.py
└── README.md
```

---

## ▶ How to Run

1. Install **Python 3**
2. Download or clone this project
3. Open terminal in the project folder
4. Run:

```bash
python calculator.py
```
